# 爬虫模块说明

## get_cookie.py — 获取登录信息
- 模拟浏览器打开微博网页版
- 人工扫码完成登录
- 读取登录后的浏览器 Cookies，保存至 `cookie.json`

---

## community_crawler.py — 爬取超话帖子
| 参数/方法 | 说明 |
|----------|------|
| `base_url` | 超话地址，注意末尾是否包含 `&page=` |
| `page` | 指定时间范围所在的页数，从该页向第一页依次爬取 |
| `scroll_page()` | 滚动页面至底部 |

- 当前仅保存 **发表时间** 和 **帖子链接**
- 也支持爬取话题下的帖子，仅需修改对应节点的 `CLASS_NAME`

---

## post_crawler.py — 爬取帖子详细内容
| 方法 | 说明 |
|------|------|
| `scroll_page()` | 滚动至底部，用于爬取点赞、评论、分享用户信息；若不需要可移除调用以节省时间 |
| `precise_scratch()` | 先爬取帖子主要内容，再调用 `scroll_page()` 获取互动用户信息 |

---

## hashtag_crawler.py — 爬取话题信息
| 参数/方法 | 说明 |
|----------|------|
| `base_url` | 含关键词的话题搜索链接 |

- 保存信息包括：**话题名称**、**话题地址**、**讨论度** 等
